┌────────────────────────────────────────────────────────────────────────────┐
│ 🎓 ESTRUTURA TEÓRICA                                                       │
└────────────────────────────────────────────────────────────────────────────┘

TRANSFER LEARNING:
  Pega modelo pré-treinado (ResNet152 em ImageNet)
  └─→ Adapta para novo problema (ciclones)
      └─→ Reutiliza features já aprendidas
          └─→ Resultado: Mais rápido e melhor!

RESNET152:
  • 152 camadas convolucionais profundas
  • Treinada em 1.2 milhões de imagens naturais
  • Já sabe reconhecer: bordas, formas, padrões, conceitos
  • Nós aproveita isso para detectar padrões de ciclones

POR QUE FUNCIONA PARA CICLONES:
  • Ciclones têm padrão espiral → ResNet já reconhece
  • Ciclones têm olho central → ResNet já reconhece
  • Ciclones têm bandas radiantes → ResNet já reconhece
  • Ciclones têm simetria → ResNet é invariante a rotações

IMPLEMENTAÇÃO:
  1. Carregamos ResNet152 com pesos do ImageNet
  2. Congelamos todos os 152 camadas (não treinam)
  3. Adicionamos 3 camadas novas (Dense) customizadas
  4. Treinamos APENAS as 3 camadas novas
  5. ResNet já treinada fornece features de qualidade

│ 📚 RESUMO DOS CONCEITOS-CHAVE                                              │
└────────────────────────────────────────────────────────────────────────────┘

Transfer Learning:
  Usar modelo treinado em dados grandes
  └─→ Adaptar para problema específico
      └─→ Resultado: Mais rápido + Melhor

ResNet152:
  Rede com 152 camadas
  └─→ Treinada em 1.2M imagens (ImageNet)
      └─→ Conhece padrões visuais gerais

Congelamento (Freeze):
  Não atualizar pesos da ResNet
  └─→ Apenas head novo aprende
      └─→ Rápido + Seguro

Fine-tuning:
  Descongelar ResNet + treinar com LR muito baixa
  └─→ Pode melhorar em mais 5-10%
      └─→ Mais lento

Ensemble:
  Múltiplos modelos + média das predições
  └─→ Resultado mais robusto
      └─→ Reduz variância
